# Desi & Dutch — Redesign Concepts

**Pakistani soul, Dutch comfort** — Utrecht

This repository contains high-fidelity static prototypes and brand assets from a design exploration session.

## What's inside

- `desi-dutch-redesign.html` — Complete restaurant/café experience  
  - Cinematic intro with animated logo (uses `logo-motion.mp4` + alpha webm)  
  - Elegant typography (Fraunces + Inter + Noto Naskh Arabic)  
  - Bilingual support (English / اردو)  
  - Full navigation, menu, gallery, story, location/booking sections  
  - Responsive + mobile menu + sticky CTA

- `dutchgym-redesign.html` — 24/7 fitness concept ("Dutch Gym")  
  - Dark brutalist/modern aesthetic with volt accent  
  - Strong typography (Archivo Black)  
  - Membership, classes, trainers, facilities

## Assets (`assets/`)

Food photography, primary logo (PNG), motion logo videos for hero/intro, and supporting imagery.

Additional reference images at root (`f*.jpg`, `t*.jpg`, `check50.png`).

## How to view

1. Clone or download
2. Open `desi-dutch-redesign.html` (or the gym one) directly in Chrome / Edge / Firefox
3. All assets load from relative `./assets/` paths — no build step needed

## Tech

Pure HTML + Tailwind-free hand-crafted CSS + vanilla JS. Self-contained prototypes.
